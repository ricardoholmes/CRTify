# CRTify

Adds a CRT filter to the screen in realtime and applies a CRT shader to images and videos.

## How to Run

`python3 main.py`

## Dependencies

- Python 3.10+
- Libraries:
    - opencv-python
    - moderngl
    - numpy
    - Pillow
    - pedalboard
    - tkinter
    - scipy
    - librosa
    - ffpyplayer
- ffmpeg
- (For real-time audio) https://vb-audio.com/Cable/
